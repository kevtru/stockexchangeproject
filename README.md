Automated Stock Exchange

This stock exchange simulator project was for a group class assignment that allows users to create an account, place bids and asks while storing, and viewing all accounts using clean object-oriented design.  
It focuses on data structures, OOP fundamentals, and creating an interactive user menu.

# Features
User actions:
- create an account
- place a bid
- place an ask
- view all accounts
- return to menu after actions are completed

# Order handling
 - stock name
 - price
 - amount
 - ID

# Data structures
- storing multiple bids and asks using lists/dictionaries

# Object-oriented classes
- 'Bidding' class
- 'Asking' class

# Tech stack
- Python 3
